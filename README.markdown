## Description
Shell script to download music from http://www.soundcloud.com.
Supports multiple pages.

## System requirements
* Unix like OS (Linux, Mac OS X...)
* curl

## Instructions
1. apply executable permissions ```chmod +x ./tweet.sh```
2. usage: ```./tweet.sh "Status"```
